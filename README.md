Tic Tac Toe
===========

A Tic Tac Toe game written in Elm.

Running the App
---------------

Make sure you have elm and elm-test installed

```bash
npm install -g elm elm-test
```

Install dependencies

```bash
elm package install
```

Run the tests

```bash
elm test
```

Build the project

```bash
elm make src/Main.elm --warn
```
